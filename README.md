# Gibs-
O Gibs+ é um projeto que desenvolvi como parte da disciplina de Desenvolvimento do 1º ano do Germinare Tech. A proposta foi criar uma página que unisse o tema de filmes com um portfólio pessoal, refletindo meus gostos, aprendizados e estilo como desenvolvedora. A aplicação apresenta um destaque de filme e seções como “Favoritos de Gigibs” e “Recomendados por Gigibs”, com pôsteres, títulos, anos e links diretos para trailers no YouTube. Além disso, inclui um sistema simples de login e logout utilizando o armazenamento local do navegador. O projeto foi construído com HTML, CSS e JavaScript, aplicando conceitos de design responsivo, organização de código e experiência do usuário, representando uma parte importante do meu desenvolvimento na área de front-end.
